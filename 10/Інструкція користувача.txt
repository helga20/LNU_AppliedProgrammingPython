WeatherLoader - програма, призначена для отримання та збереження погодових даних міста Львів з використанням OpenWeatherMap API.  
WeatherStatictics - програма, призначена для обробки та аналізу даних погоди, зібраних з різних файлів у форматі JSON. Вона виконує декілька операцій з аналізу та виводу статистики на основі цих даних.


Роботу виконали:

Ласько Маркіян (ПМО-41): 
	1) розробка функції GetServerAddress(apiKey), яка створює URL для OpenWeatherMap API на основі переданого API ключа;
	2) реалізація функції GetDataFromServer(), яка перевіряє існування файлу з погодовими даними на поточну дату, встановлює з'єднання з сервером OpenWeatherMap, отримує віддалений файл та зберігає його локально.

Кравець Ольга (ПМО-41):
	1) створення функції GatherAllData(result_file), яка аналізує структуру JSON усіх .json файлів і виводить основну інформацію у вказаний файл result_file;
	2) реалізація функцій для отримання, обробки та аналізу значень збережених даних за конкретним ключем (GetValuesByKey, GetValueByKey, GetMaxValueByKey, GetAvgValueByKey, GetDaysWithParticulatWeatherCondition, GetDaysWithValueMoreThan).

Кравець Назар (ПМО-41):
	1) реалізація підпроцесу для взаємодії з мовою C# (Windows Forms App) в межах Python-проєкту;
	2) реалізація функцію runSubprocess(), яка викликає підпроцес на C#, запускаючи зазначений виконавчий файл.


----------- Інструкція користувача для використання програм -----------

WeatherLoader.py:
	1. Запустіть WeatherLoader.py.
	2. Програма підключиться до сервера OpenWeatherMap, отримає поточні погодові дані для міста Львів та збереже їх у файл JSON із назвою, що відображає поточну дату.
	3. Перевірте консольний вивід для інформації щодо завантаження та збереження файлу.

WeatherStatistics.py:
	1. Запустіть WeatherStatistics.py.
	2. Програма проаналізує структуру погодових даних, збережених у файлах JSON та виведе інформацію в файл result.txt.
	3. У файлі result.txt ви знайдете основну інформацію про структуру погодових даних, значення максимальної температури, середню вологість, дні з певними умовами погоди, а також дні з вологістю більше 98%.
	4. В консолі буде виведено повідомлення про запуск та завершення підпроцесу на мові C#.

! Важливо:
	- Переконайтеся, що у вас встановлені всі необхідні бібліотеки, такі як urllib.
	- Перевірте шляхи до виконавчих файлів (наприклад, script.exe), і впевніться, що вони доступні для виконання.
	- Відкрийте WeatherLoader.py та вставте ваш API-ключ OpenWeatherMap в рядок remoteaddr = GetServerAddress('1952efb4a7da640d955fd67682858a7f').
	- Забезпечте відповідність шляхів до каталогів та файлів у вашому середовищі виконання.
